# typescript-bootstrap

A bootstrapping Repo for Typescript projects

## Building
If you would like to do a full clean build:

`make cleanBuild`

***Note:** it is good practice to run this any time you switch or update a branch to make sure you do not have any unexpected artifacts.*

**To do incremental builds:**

`make build`

**To clean your environment:**

`make clean`

## Local Testing
**To run just Unit tests:**

`make unitTest`

**To run just Integration Tests:**

`make integrationTest`

**To run the Full test suite:**

`make test`

## Linting
**To run the linter:**

`make lint`

## Semantic-Releas
This project usses semantic-releasee

## Commit checks
**to run the commit check:**

`make lintCommit`
